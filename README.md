# 🕯️ PrayCircle

**United in prayer, around the world.**

PrayCircle is a Catholic prayer community app. Users post short prayer
intentions ("please pray for my history exam", "pray for my grandmother in
hospital") and others tap **Pray** to receive a prayer perfectly fitted to
that intention — then seal it with **Amen**, which quietly notifies the
person who asked.

## Features

- ✝️ **Prayer intentions feed** — short, moderated posts (max 180 characters)
- 🙏 **Fitting prayers** — an engine detects what an intention is about
  (exams, illness, grief, family, work, travel, addiction, peace, …) and
  serves a fitting Catholic prayer in the reader's language; with an
  `ANTHROPIC_API_KEY` set, bespoke prayers are generated by Claude
- 🕊️ **Amen → notification** — the author is notified every time someone
  prays for their intention
- 🌍 **Three languages** — English, German, Italian, with one-tap toggle and
  post translation
- 🛡️ **Moderation** — hateful, racist, violent, profane and spam content is
  rejected before it is ever published
- 👤 **Accounts** — email + password sign-up, unique usernames
  (case-insensitive), self-service account deletion (GDPR)
- 📱 **PWA** — installable, offline shell, splash screen with rotating
  phrases (never the same twice in a row)
- ⚖️ **Legal** — privacy policy, terms of service and cookie policy drafted
  for Italian law (GDPR, Codice Privacy, Garante cookie guidelines) in all
  three languages

## Stack

- [Next.js 15](https://nextjs.org) (App Router) + React 19 + TypeScript
- [Turso](https://turso.tech) (libSQL) — falls back to a local `dev.db`
  SQLite file when `TURSO_DATABASE_URL` is not set
- JWT sessions (`jose`) in httpOnly cookies, `bcryptjs` password hashing
- Deployed on [Vercel](https://vercel.com), auto-deploys from `main`

## Environment variables

| Variable | Required | Purpose |
| --- | --- | --- |
| `TURSO_DATABASE_URL` | production | libSQL URL of the Turso database |
| `TURSO_AUTH_TOKEN` | production | Turso database auth token |
| `AUTH_SECRET` | production | secret for signing session JWTs |
| `ANTHROPIC_API_KEY` | optional | bespoke AI prayers + higher-quality translation |

## Development

```bash
npm install
npm run dev
```

No configuration needed locally — a `dev.db` SQLite file is created
automatically. To regenerate the PWA icons: `npm i --no-save sharp && node
scripts/generate-icons.mjs`.

## Turso setup

```bash
turso db create praycircle
turso db show praycircle --url        # → TURSO_DATABASE_URL
turso db tokens create praycircle     # → TURSO_AUTH_TOKEN
```

Tables are created automatically on first request.
